# Time Series Analysis — Exercises

Personal code recreation of the *Time Series Analysis Handbook* by [phdinds-aim](https://github.com/phdinds-aim/time_series_handbook).

This repository contains my own independently worked-through implementations of 
the concepts and exercises covered in the Preface chapter, written for the purpose 
of learning.

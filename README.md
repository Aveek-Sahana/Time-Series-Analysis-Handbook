# Time Series Analysis — Exercises

Personal code recreation of the *Time Series Analysis Handbook* by [phdinds-aim](https://github.com/phdinds-aim/time_series_handbook).

This repository contains my own independently worked-through implementations of 
the concepts and exercises covered in the Preface chapter, written for the purpose 
of learning.

## About the Source Material

The Time Series Analysis Handbook is an open-source educational resource covering 
time series methods from foundational statistical models to state-of-the-art 
algorithms, implemented in Python.

- 📖 Full Handbook: https://phdinds-aim.github.io/time_series_handbook/
- 💻 Original Repository: https://github.com/phdinds-aim/time_series_handbook
